# 📈 Stock Market Simulator — V1

A virtual stock trading platform built .
Users start and can buy/sell certain Indian stocks.

## Tech Stack
- **Frontend:** React, React Router, Axios
- **Backend:** Node.js, Express, Socket.io (V2)
- **Database:** MongoDB Atlas + Mongoose
- **Auth:** JWT + bcryptjs

## Features (V1)
- [x] JWT Signup / Login
- [x] Virtual wallet ₹1,00,000
- [x] Buy & Sell stocks with balance validation
- [x] Portfolio dashboard with P&L
- [x] Transaction history


it handles all those efficiently
